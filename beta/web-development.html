Web Development

Front-end web development is programming that instructs the visitor's browser how to display, read or present the web content. All web sites require front-end coding (programming). Top quality front-end web programming includes pure CSS layouts, semantic XHTML, ethical search engine optimization, unobtrusive JavaScript, "Best Practices", and strict adherence to current web standards provided by the World Wide Web Consortium (W3C). I create error-free (valid) Strict XHTML that meets the most rigorous W3C Guidelines.

Here is a partial list of benefits my clients experience as a result of my strict adherence to W3C-standards-compliant web coding:

  • Improved search results
  • Lower maintenance costs
  • Consistent appearance on most browsers
  • Reduced bandwidth costs
  • Faster page loading

My Coding Capabilities

  • XHTML/CSS. I craft beautiful code that is both valid and elegant. Clean code loads faster and more efficiently, giving my websites a competitive edge.

  • Javascript/AJAX. After a website has a solid XHTML/CSS foundation, Javascript allows me to improve upon the interactivity of a page and make it feel more user-friendly. This can range from subtle mouse-over effects to the complete loading and submitting of forms in the background without any visible loading of the page. My favorite library is jQuery.

  • PHP/MySQL. PHP and MySQL are the most standard web development technologies and I do most of my custom development with them. Because they are so standard, an application developed with PHP and MySQL can be dropped on almost any server and run consistently.

  • Flash Actionscript. ActionScript is the native scripting language of Flash. I use ActionScript to create interactive Flash websites, Flash banner ads, and presentations.

  • Content Management Systems. I develop websites on top of robust content management systems. This allows you to make basic changes to your website at any time without having to wait days or weeks for a webmaster to assist you.
